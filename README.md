### Usage

```
npm install
npm start
open http://localhost:3000
```
Also hosted on Heroku: https://slackr-chat.herokuapp.com/

### Dependencies

* React
* Webpack
* [babel-loader](https://github.com/babel/babel-loader)
* [webpack-dev-server](https://github.com/webpack/webpack-dev-server)
